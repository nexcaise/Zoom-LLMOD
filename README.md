# Zoom

A simple Zoom mod for Minecraft Bedrock Edition (Windows).

## Features

- Press 'C' to zoom in.
- Supports v1.26.20.04
- Smooth zoom animation.

## Requirements

- [LeviLauncher (GDK)](https://github.com/LiteLDev/LeviLauncher)

## Installation

1. Download `Zoom.dll` from [Releases](https://github.com/RadiantByte/LeviLauncher-Zoom/releases) or GitHub Actions.
2. Install LeviLauncher.
3. Add the `Zoom.dll` mod in LeviLauncher.
4. Launch Minecraft through LeviLauncher.
